Hi, I'm Andrii.

I work on machine learning for mobile applications, mainly in computer vision and on-device AI. My interests include practical ML systems, model optimization, and the engineering challenges that arise between a trained model and a production feature.

This blog documents projects, experiments, failures, and lessons learned from building machine learning systems. Along the way, I also write about performance optimization, efficient inference, numerical computing, and other engineering problems that I find interesting.
